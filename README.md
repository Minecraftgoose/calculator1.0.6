# 小鹅计算器 

一个功能强大、响应式的在线科学计算器应用，专为iPad和现代浏览器设计。

## 功能特点
- 基础四则运算
- 科学计算（sin、cos、tan、cot）
- 百分比计算
- 横竖屏自适应
- 深色/浅色主题
- 个性化设置
- 离线使用（PWA）
- iPad优化

## 在线访问
👉 [https://minecraftgoose.github.io/calculator1.0.6/](https://minecraftgoose.github.io/calculator1.0.6/)

## 技术栈
- HTML5, CSS3, JavaScript
- 响应式设计
- PWA支持
- 本地存储
